## Installation

Download the latest release version directly from Github.

OR

```
git clone https://github.com/MrScalo/FindFire.git
cd FindFire
```

To run the slower development version...

```
cargo run
```

For the fast release version...

```
cargo build --release
./target/release/FindFire
```
